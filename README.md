# Light of Mine

A 3D open-world RPG inspired by Morrowind (Elder Scrolls), developed for mobile and web browsers using Three.js.

## Vision

Phase 1 is to develop an Island region/demo area with as many features and mechanics as possible. The core idea is a simple system and workflow for building the game world using JSON objects that define and arrange 3D assets into scenes — terrain, structures, NPCs, items, triggers, etc.

## Tech Stack

- **Rendering:** Three.js
- **Platforms:** Mobile & web browsers
- **Scene System:** JSON-driven world/area definitions
